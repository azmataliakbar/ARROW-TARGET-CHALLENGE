# Arrow Target Challenge

A fast-paced target-shooting game built with vanilla HTML, CSS, and JavaScript on Canvas. Aim your bow, hit moving targets, build combos, and climb the leaderboard.

## How to Play

- Shoot arrows at the pulsing target to score points
- Each hit earns **10 base points**, multiplied by your combo streak
- **Missing a target costs you one arrow** — run out and it's game over
- **Hit 3 targets in a row** to earn a bonus arrow (up to 10 max)
- The target gets smaller and faster as your score increases

## Controls

| Action | Desktop | Touch |
|--------|---------|-------|
| Move | Arrow keys or WASD | On-screen buttons |
| Shoot | Space or mouse click | SHOOT button |
| Reset | RESET button | RESET button |

## Features

- Combo scoring system with on-screen multiplier
- Screen shake and particle effects on hits
- Sound effects (shoot, hit, button click) via Web Audio API
- Responsive layout from 270px mobile screens up to desktop
- Touch controls for mobile / tablet
- High score saved to `localStorage`

## Play It

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Click **NEW GAME** and start shooting

## Tech

- HTML5 Canvas
- CSS3 (Flexbox, animations, responsive breakpoints)
- Web Audio API (procedural sound effects)
